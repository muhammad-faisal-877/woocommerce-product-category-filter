# 🛍️ WooCommerce Product Category Filter

Filter WooCommerce shop and search results to display only selected product categories.

## ✨ Features

- Filter shop page products
- Filter search results
- Admin category selection (Select2)
- Lightweight & fast

## ⚙️ Usage

- Go to Settings → Product Filter
- Select categories
- Save

## 🧠 Use Case

Useful for:
- Multi-store setups
- B2B filtering
- Restricted catalogs
